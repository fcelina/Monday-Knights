# Monday Knights Website - PRD

## Original Problem Statement
User requested to pull their Monday Knights website from GitHub repository (https://github.com/fcelina/Monday-Knights) to deploy with Emergent. The website was originally built with Emergent on a separate account.

## Architecture
- **Frontend**: React with Tailwind CSS, CRACO config
- **Backend**: FastAPI with MongoDB
- **Database**: MongoDB (local)

## User Personas
- **Visitors**: Community members viewing About Us, Contact forms, Events
- **Admin**: Site administrator managing content, blog posts, contact submissions

## Core Requirements
- Display Monday Knights branding and artistic design
- About Us, Contact Us, Events overlays
- Contact forms (Individual & Business)
- Admin panel for content management
- Blog/Events posting system
